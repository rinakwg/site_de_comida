# 🍽️ FoodHub - Site de Comida Estilo iFood

Um site de delivery de comida moderno, responsivo e divertido com uma aba especial para "seu docinho"!

## 🎨 Características

✨ **Design Moderno e Responsivo**
- Layout responsivo que funciona em desktop, tablet e mobile
- Cores vivas e vibrantes (vermelho, amarelo, verde e azul)
- Animações suaves e transições dinâmicas
- Interface limpa e intuitiva

🍣 **Seção de Sushi**
- 4 restaurantes de sushi premium
- Cards informativos com avaliações e preços
- Interface visual atraente com gradientes

🍕 **Seção de Pizza**
- 4 pizzarias diferentes com diversas especialidades
- Informações de entrega e valores
- Cards interativos

💋 **Aba "Seu Docinho" (Especial)** 
- Uma surpresa fofa para zoar com seu namorado/namorada 😏
- Animações fofas com emojis flutuantes
- Card especial com avaliações divertidas
- Efeito de corações quando clica em "Chamar Agora"

🔍 **Funcionalidades Interativas**
- Busca de restaurantes em tempo real
- Navegação entre categorias com transições suaves
- Notificações customizadas ao fazer pedidos
- Easter eggs e animações especiais

## 📱 Responsividade

- ✅ Desktop (1200px+)
- ✅ Tablet (768px - 1200px)
- ✅ Mobile (até 480px)

## 🚀 Como Usar

1. **Abra o arquivo `index.html` em seu navegador**
   - Clique duplo no arquivo ou arraste para o navegador

2. **Navegue entre as categorias**
   - Clique nos botões "🍣 Sushi", "🍕 Pizza" ou "💋 Seu Docinho"

3. **Busque por restaurantes**
   - Digite o nome na barra de busca e clique no ícone de lupa

4. **Faça pedidos**
   - Clique em "Pedir Agora" para adicionar ao carrinho
   - Aproveite as notificações personalizadas!

5. **Descubra a surpresa**
   - Clique em "Seu Docinho" e depois em "👉 Chamar Agora"
   - Observe a magia acontecer ✨

## 🎯 Funcionalidades do Script

- **Navegação entre categorias**: Alerna entre seções sem recarregar
- **Busca inteligente**: Procura restaurantes por nome
- **Notificações**: Feedback visual ao fazer pedidos
- **Easter egg especial**: Corações animados para "seu docinho"
- **Responsive design**: Adapta-se a qualquer tamanho de tela
- **Animações suaves**: Transições elegantes entre seções

## 🔧 Customização

Para personalizar o site:
- **Cores**: Edite as variáveis CSS em `:root` no `styles.css`
- **Restaurantes**: Adicione novos cards no HTML
- **Textos**: Modifique o conteúdo no `index.html`
- **Animações**: Ajuste as keyframes no `styles.css`
